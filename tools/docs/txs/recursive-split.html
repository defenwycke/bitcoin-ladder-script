Recursive Split (Tree)

Binary split covenant: a UTXO splits into 2 equal children on each spend, up to 3 levels deep. Each child inherits the same rung conditions with max_splits decremented. When splits are exhausted, the UTXO becomes freely spendable. This creates an on-chain binary tree of UTXOs from a single parent.

RECURSE_SPLIT · SIG
Split Tree Structure

Each spend divides the UTXO into two equal children with identical conditions. The max_splits counter decrements at each level, preventing infinite recursion.

How Recursive Splitting Works
1.
Funding transaction creates a rung output with SIG + RECURSE_SPLIT blocks. The RECURSE_SPLIT block stores max_splits (remaining split depth) and min_sats (minimum sats per child).
2.
When spending, the evaluator checks max_splits > 0. If so, the covenant requires the transaction to produce exactly 2 outputs, each with half the remaining sats (after fee) and identical rung conditions.
3.
The child outputs' conditions are the same as the parent, except max_splits is decremented by 1. This is enforced at consensus — the spender cannot change the conditions or the split counter.
4.
When max_splits reaches 0, the RECURSE_SPLIT block is SATISFIED unconditionally — the leaf UTXO is freely spendable (only the SIG block remains as a condition).
5.
The min_sats guard prevents the tree from creating dust outputs. If half < min_sats, the split is rejected — the UTXO must accumulate more value or be spent as a leaf.
Use Cases
ApplicationDescription
Payment Distribution A single funded UTXO fans out to 2N recipients through a binary tree of on-chain splits, each requiring only a signature.
CoinJoin Trees Pre-committed split structure where each participant controls a branch. Tree shape is fixed at funding time.
Vaults with Granular Withdrawal Large vault UTXO splits into smaller chunks on demand. Each chunk inherits the vault's security conditions (timelocks, multisig).
Post-Quantum Tree Combined with Falcon-512/1024 SIG blocks, creates a PQ-secured binary tree where every split inherits quantum resistance.
