Quantum Vault + Children

Post-quantum secured vault using Falcon-512 signatures that splits into child UTXOs via RECURSE_SPLIT. Each child inherits the same PQ protection. Four spend paths: split (binary tree), child leaf (amount-locked), emergency multisig, and CSV timeout sweep with Dilithium3.

FALCON-512 · RECURSE_SPLIT · DILITHIUM3
Rung Structure

Four rungs provide four spend paths. PQ_SPLIT creates a binary tree of child UTXOs. PQ_CHILD provides an amount-locked leaf path. PQ_EMERG is a 2-of-2 Falcon multisig emergency exit. SWEEP is a Dilithium3-signed CSV timeout recovery.

The PQ_SPLIT rung creates a binary tree. Each spend divides the UTXO into two equal children with max_splits decremented. Every child inherits Falcon-512 quantum resistance. At max_splits=0, the leaf is freely spendable.

How Quantum Vault Splitting Works
1.
Funding transaction creates 4 rung outputs. The PQ_SPLIT rung contains a Falcon-512 SIG block and a RECURSE_SPLIT block with max_splits=3 and min_sats=10,000.
2.
To split, the PQ_SPLIT rung is spent. The evaluator verifies: (a) valid Falcon-512 signature, (b) max_splits > 0, (c) exactly 2 outputs with ≥ min_sats each, and (d) both children carry the same conditions with max_splits − 1.
3.
Each child UTXO inherits the full PQ protection — Falcon-512 key commitment, split parameters, and all alternative spend paths (PQ_CHILD, PQ_EMERG, SWEEP). The quantum resistance is preserved through every level of the tree.
4.
When max_splits reaches 0, the RECURSE_SPLIT block is SATISFIED unconditionally — the leaf UTXO is freely spendable with just a Falcon-512 signature. The tree produces up to 23 = 8 leaf UTXOs.
5.
Alternative paths: PQ_EMERG requires a 2-of-2 Falcon multisig for emergency recovery. SWEEP requires a Dilithium3 signature after a 4,320-block CSV delay (~30 days), providing a secondary PQ scheme as backup.
Use Cases
ApplicationDescription
PQ Treasury Distribution Large quantum-resistant treasury fans out to 2N recipients through binary splits. Every branch is protected against quantum attack throughout the distribution.
Hybrid PQ Recovery Primary spend via Falcon-512; timeout recovery via Dilithium3. Two independent PQ schemes ensure funds are recoverable even if one scheme is compromised.
Granular Vault Withdrawal Vault splits into progressively smaller chunks on demand. Each chunk inherits the full security model — emergency multisig, CSV timeout, and PQ signatures.
Post-Quantum CoinJoin Tree Pre-committed quantum-resistant split tree where each participant controls a branch. Tree structure is fixed at funding time with Falcon-512 key commitments.
