2-of-3 Multisig Vault

Multi-output vault pattern using the MULTISIG block type. A v4 RUNG_TX creates three outputs: a 2-of-3 threshold multisig for primary spending, a timelocked single-key recovery path (CSV 144 blocks), and change. The spend demonstrates satisfying the 2-of-3 threshold with two Schnorr signatures from two of the three committed keys.

MULTISIG · TYPE 0x01 CSV · TYPE 0x03 SCHNORR · SCHEME 0x01
Output Structure

The funding transaction splits funds across three outputs with different security policies. The primary output requires any 2-of-3 signers. The recovery output adds a 144-block (~1 day) CSV delay as a time-gated escape path. Change returns to a single key.

The MULTISIG block stores three 32-byte PUBKEY_COMMITs (SHA-256 hashes of compressed public keys) and a NUMERIC threshold of 2. The SCHEME field selects Schnorr signatures. At spend time, 2 of 3 keys are revealed in the witness alongside their signatures.

How 2-of-3 Multisig Spending Works
1.
Funding transaction creates three outputs. Vout 0 carries rung_conditions with a single MULTISIG block: NUMERIC threshold = 2, three PUBKEY_COMMITs (SHA-256 hashes of each signer's compressed public key), and SCHEME = SCHNORR.
2.
To spend, 2 of the 3 keyholders construct a transaction. The ladder witness contains 4 explicit fields: PUBKEY (key 1), SIGNATURE (sig 1), PUBKEY (key 2), SIGNATURE (sig 2). Fields are paired: each key is followed by its signature.
3.
The evaluator reads the threshold (2) from the conditions, then iterates the witness fields. For each PUBKEY, it computes SHA256(pubkey) and verifies the hash matches one of the three PUBKEY_COMMITs. For each SIGNATURE, it validates against the Ladder sighash using the preceding key. Result: 2 valid signatures from distinct committed keys ≥ threshold. SATISFIED.
4.
Vout 1 provides a recovery path: after 144 blocks (~1 day CSV delay), key 1 alone can sweep the recovery output. This is a separate UTXO, not an alternative rung — funds are split at funding time so each path controls its own amount.
